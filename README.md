# Hidden Markov Models

### Prerequisites
Python 2.7 or 3.4 or greater  
numpy>=1.14.0  
tabulate>=0.8.2  
pytest>=3.5.0  

Install with 
```bash
cd hmm
pip install -r requirements.txt
```

### Tests

Execute ```pytest``` in the hmm directory

### Examples

Execute ```python example_NCSA.py``` or ```python example_Gene.py``` in the hmm directory.
